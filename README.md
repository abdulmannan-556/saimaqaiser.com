## Overview

This repository contains the fully upgraded, multi-page website for Saima Qaiser Securities (Pvt) Ltd.  
The website includes:

- Home, About, Services, Governance, Investor Information, Investors Portal, Feedback, KYC/CDD, and Contact pages  
- Dropdown menus and interactive components  
- Stock market ticker on the homepage  
- Mobile-responsive design  
- Animations and transitions for professional UI  

**All content and frontend developed by Abdul Mannan Butt.**
